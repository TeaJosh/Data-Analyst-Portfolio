# Car Sales Analysis

## Description
Analysis of **23,906 automotive transactions** across **7 regions** to identify pricing drivers, customer patterns, and dealer performance using statistical testing and data visualization.

Statistical methods include **correlation analysis, chi-square tests, and t-tests** to uncover meaningful relationships in the data.

---

## Tools
- R  
- RStudio  
- tidyverse  
- ggplot2  

---

## Skills Demonstrated
- Data cleaning  
- Feature engineering  
- Statistical analysis  
- Hypothesis testing  
- Customer segmentation  
- Time series analysis  
- Data visualization  

---

## Key Findings
- Gender significantly influences vehicle preferences (**p < 0.05**)  
- Automatic transmissions command higher prices than manual transmissions  
- Weak correlation between income and vehicle price (**r ≈ 0.1–0.3**)  
- Regional performance disparities identified across **7 markets**  

---

## Dataset
- **Source:** Kaggle  
- **Size:** 23,906 rows × 16 columns (21 columns after feature engineering)  
- **Date Range:** January 2022 – December 2023  
- **Price Range:** \$3,750 – \$85,000  
- **Income Range:** \$13,500 – \$4.7M+
